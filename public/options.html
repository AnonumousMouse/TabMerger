Exclude Websites:
Please enter the full website address of any website which you would like to not merge.
These must begin with "http://" or "https://" - we recommend copying the link of the tab and pasting it here.
You may enter multiple comma seperated values.
Note: TabMerger provides a menu for you to quickly do this, simply right click on any page you want to exclude, then in the context menu popup navigate to:
TabMerger > Exclude Site from TabMerger's Visibility
Doing this will cause the corresponding tab's URL to show up here right away, without the need to navigate to this settings page.
Extension Opening Behavior:
The following happens when TabMerger's extension icon is clicked.
This will open (or switch if already open) TabMerger's extensions page without merging the currently open tabs in the same window. The user can then merge open tabs by using each group's merging buttons. Such a behavior is desireable for full control of the merging process.
This will open (or switch if already open) TabMerger's extensions page AND merge the currently open tabs in the same window. The merged tabs will be grouped into the top group. Thus, the user will then be able to sort the tabs along various groups via the drag-and-drop feature. This behavior is desireable when the user wants to avoid opening an extra tab (TabMerger) prior to merging.
Restoring Behavior:
This applies to both group and global actions.
You can still manually delete:
  • tab → ⨯ button beside them (on the left)
  • group → Delete Group button (inside the group)
  • everything → global Delete All button
Opening a tab (Click on it) will remove that tab from TabMerger's page, the same goes for opening tabs in a group (Open Group button) and opening all tabs (Open All button). Note: in the Open Group case, the corresponding (ONE) group will be removed. Likewise, in the Open All case, ALL groups will be removed.
