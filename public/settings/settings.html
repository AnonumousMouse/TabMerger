If you opt to randomize group colors, a randomly selected color from a predefined list will be assigned to newly formed groups. Color randomization only applies to users with Premium subscription.
Default Group Title:
Applied to new groups formed after settings are saved!
Global Font Family:
Applied to everything in TabMerger directly upon saving! Only applies to users with Standard or Premium subscription.
Tab Title Font Weight:
Applied only to the tab titles directly upon saving! Only applies to users with Standard or Premium subscription.
Exclude Websites:
Please enter the full website address of any website which you would like to not merge. This URL must begin with "http://" or "https://" - we recommend copying the link of the tab and pasting it here. You may enter multiple comma seperated values (include a space after each comma).
Note: TabMerger provides a menu for you to quickly do this, simply right click on any page you want to exclude, then in the context menu popup navigate to:
TabMerger > Exclude Site from TabMerger's Visibility
This places the corresponding tab's URL here right away, without the need to navigate to this settings page.
Automatic Backup:
Only applies to users with Premium subscription. Automatically generates JSON configuration files relative to the Downloads folder. Will automatically sync (write) your TabMerger configuration.

Relative Path (JSON):

This is a path relative to the Downloads directory where the JSON file will be saved. For example, "TabMerger/" would generate files in "Downloads/TabMerger/". Must not contain "../", "./", or absolute paths!

Save JSON Every ___ Minute(s):

Sync Every ___ Minute(s):

For both JSON & Sync backups: minimum 0 minutes (default - disable automatic backup), maximum 1 day (1440 minutes).

Maximum JSON Files To Backup:

Minimum: 1, maximum: 100, default: 15. When reached, older files are deleted to avoid producing too many backup files.
