Default Group Color:
This will only be applied on new groups formed after settings are saved!
Default Group Title:
This will only be applied on new groups formed after settings are saved!
Exclude Websites:
Please enter the full website address of any website which you would like to not merge. This URL must begin with "http://" or "https://" - we recommend copying the link of the tab and pasting it here. You may enter multiple comma seperated values.
Note: TabMerger provides a menu for you to quickly do this, simply right click on any page you want to exclude, then in the context menu popup navigate to:

TabMerger > Exclude Site from TabMerger's Visibility

Doing this will cause the corresponding tab's URL to show up here right away, without the need to navigate to this settings page.
Badge Appearance:
Whether or not information is displayed on TabMerger's extension badge.
Shows relevant information about number of tabs and groups inside TabMerger at any moment. Shown as {#tabs}|{#groups} with a background color corresponding to number of tabs. The user can hover over the icon to see these numbers in a clear sentence.
No extra information is displayed on top of the badge icon. However, the user can still hover over the badge icon to see the relevant tab & group information.
Extension Opening Behavior:
The following happens when TabMerger's extension icon is clicked.
This will open (or switch if already open) TabMerger's extensions page without merging the currently open tabs in the same window. The user can then merge open tabs by using each group's merging buttons. Such behavior is desireable for full control of the merging process.
Same as above but with merging. The merged tabs will be grouped into a new group. Thus, the user will then be able to sort the tabs among various groups or simply sort the groups via the drag-and-drop feature. This behavior is desireable when the user wants to avoid opening an extra tab (TabMerger) prior to merging.
General Merge Behaviour:
This is considered when merging any tab into TabMerger.
Will close any merged tabs to reduce the load on your system. These tabs can be accessed from TabMerger at any time.
Will not close merged tabs while still storing them in TabMerger. This is desireable when the user would like to continue working on the current tab(s) but wants to ensure that they are saved in TabMerger for restoration at a later time.
Pinned Tab Merge Behavior:
This is considered when merging pinned tabs into TabMerger.
TabMerger also allows you to toggle the tab's pin status directly.
Your pinned tabs will NOT be merged. Only regular tabs (unpinned) are considered in the merging process.
Restoring Behavior:
This applies to both group and global actions.
You can still manually delete tabs by the delete tab, group, or all actions.
Opening a tab (Click on it) will remove that tab from TabMerger's page. This holds for any other opening action (group or all).
