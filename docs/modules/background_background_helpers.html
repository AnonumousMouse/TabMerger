filterTabs

  • filterTabs(info: { which: string }, tab: { id: string; title: string; url: string }, group_id: string): Promise<void>
    • Defined in public/background/background_helpers.js:38

    Filters a list of tabs according to the merging information provided in the parameters. Sets the local storage item corresponding to the group to merge into and the tabs to merge. Avoids duplicates as much as possible. If duplicate tabs are merged, only a single copy of the many duplicates is included in the merge (the other duplicate tabs are simply closed).

    Parameters

    • info: { which: string }

      which direction to merge from

      • which: string
    • tab: { id: string; title: string; url: string }

      indicates where the merging call originated from

      • id: string
      • title: string
      • url: string
    • group_id: string

      the group to merge into (if merge button from one of TabMerger's groups is used)

    Returns Promise<void>

findExtTabAndSwitch

  • findExtTabAndSwitch(testing: boolean): Promise<any>
    • Defined in public/background/background_helpers.js:109

    When TabMerger is open, this navigates to its tab if not on that tab already. When TabMerger is not open, this opens its tab on the very right side. Function ends when TabMerger's tab becomes active and its loading status is complete.

    Parameters

    • testing: boolean

      Whether this is a testing call or a real call

    Returns Promise<any>

    A promise which should be awaited. Resolve value is insignificant
