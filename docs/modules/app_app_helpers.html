updateGroupItem

  • updateGroupItem(key: string, value: DefaultGroup): Promise<unknown>
    • Defined in src/components/App/App_helpers.ts:107

    Chrome's storage sync has limits per item. Therefore, the groups are stored individually as one group per item (unlike in Chrome's storage local - where they are all one big item). Due to this and the fact that there is also a limit of sync write operations per minute, Sync items are updated only when they change and only changed items are overwritten with new values.

    note

    Exported for testing purposes

    Parameters

    • key: string

      Group id of an item that might need to be updated

    • value: DefaultGroup

      Value corresponding to the above group id

    Returns Promise<unknown>

    Promise that resolves if sync is updated or no update is required
