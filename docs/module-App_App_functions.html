(static) regexSearchForTab(e)

Allows the user to search for groups or tabs within TabMerger using a filter. The filter is case-insensitive and updates the groups in real time as the user types. This action is non-persistent and will reset once the page reloads or a re-render occurs. Tabs/Groups are simply hidden from the user once a filter is applied, that is they are not removed from TabMerger and thus the counts (group and global) are not updated.
