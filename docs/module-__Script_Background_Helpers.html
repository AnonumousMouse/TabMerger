(static) filterTabs(info, tab, group_idnullable)

Filters a list of tabs according to the merging information provided in the parameters. Sets the local storage item corresponding to the group to merge into and the tabs to merge. Avoids duplicates as much as possible. If duplicate tabs are merged, only a single copy of the many duplicates is included in the merge (the other duplicate tabs are simply closed).

(static) findExtTabAndSwitch()

When TabMerger is open, this navigates to its tab if not on that tab already. When TabMerger is not open, this opens its tab on the very right side. Function ends when TabMerger's tab becomes active and its loading status is complete.
